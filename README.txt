本系统是在eclipse下仅用java语言开发的一个模拟操作系统，
用文件模拟磁盘，实现了对：
        （1）文件管理和用户接口
        （2）存储管理
        （3）设备管理
        （4）进程管理
4大功能的模拟，我们采用GitHub做为团队开发的版本控制器，
我们遵循GitHub的开源协议。
